# Byobu

[byobu](https://www.byobu.org/home)

## 특징
1. 멀티 세션 유지가 가능하다
2. 접속을 종료하거나 끈긴 이후에도 작업을 확인할 수 있다
3. 아주 단순하고 강력하다

## 설치
```bash
$ sudo apt-get install byobu
```

## 실행
```bash
$ byobu
```

## 사용법
- F2 : 새창 생성
- F3 : 왼쪽 세션 이동
- F4 : 오른쪽 세션 이동
- F6 : 세션 나가기
